# Notes App

A simple and clean web application for creating and managing notes. The app is built with HTML5, Bootstrap 5, and vanilla JavaScript.

## Features

- **Create Notes:** Add new notes with a title and content.
- **Edit Notes:** Update existing notes.
- **Delete Notes:** Remove individual notes or clear the entire list.
- **Local Storage:** All notes are saved in the browser's `localStorage`, so they persist between sessions.
- **Dark/Light Mode:** Switch between light, dark, or the system's default theme.
- **Collapsible List:** Collapse and expand the note list for a more compact view.

## How to Use

As this is a pure front-end project, no build or installation is required. Simply clone or download the source code, and then open the `index.html` file in any modern web browser.
